Detailed Description

Line struct and functions The lineStruct represents a mathematical line. It has two points (A,B) and a type (type). The type is used like an enum. It determines whether the line ends at each point, or whether the line extends to mathematical infinity.

type: (0) a line-segment (1) an infinite line (2) a line starting at B and extending (infinitely) in direction A (3) a line starting at A and extending (infinitely) in direction B
