◆ intersection()

vector intersection ( const planeStruct plane;const lineStruct line;int  success)

Returns intersection location between lineStruct and planeStruct and edits an int "success" variable WARNING: This currently does not account for different line types. Also the success results will be changed so failure will be encoded with 0 or a negative number and success will be a number greater

success: 0 => Failure, no intersection. The planeStruct and the lineStruct never intersect 1 => The plane intersects the line in the returned position 2 => Failure. The lineStruct lies in the plane (is parallel to it) 3 => The intersection lies outside the segment

Parameters
{planeStruct}{plane} a planeStruct
{lineStruct}{line} a lineStruct
{int}{success} a variable that is passed by reference (is manipulated by the function). It will return "success" (as explained above)

From: https://de.mathworks.com/matlabcentral/fileexchange/17751-straight-line-and-plane-intersection
